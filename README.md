# MileStone Tiimes website

First static ecommerce presence for MileStone Tiimes, built for GitHub Pages.

## What is included

- Responsive homepage for Ottawa custom cakes and dessert treats.
- Product/order sections for cupcakes, cake pops, rum balls, standard boxes, simple cakes, wedding cakes, and treat tables.
- Cart-style order list for quick treats.
- Quote request flow for custom cakes, wedding cakes, and treat tables.
- Build a Treat Box configurator with minimum 3-item selection and cart/order summary details.
- Checkout details flow aligned to the current payment model: e-transfer deposit and cash remainder at pickup or delivery.
- Email and Facebook DM handoff links that prepare order details for juliavalcin@gmail.com and the MileStone Tiimes Facebook inbox.
- Occasion-led shopping paths for birthdays, weddings, event tables, and gift boxes.
- Order handoff fallback panels with copyable details for email or Facebook Messenger.
- Thank-you confirmation screen after email handoff with next steps, resend links, Facebook DM, and copyable details.
- Separate client contact fields for first name, last name, email address, and phone number on both checkout and custom quote requests.
- Minimum date and notice guidance for quick treats and custom milestone orders.
- Local public assets captured from the MileStone Tiimes Facebook page for the first version.

## GitHub Pages publishing

1. Push these files to a GitHub repository.
2. In GitHub, open Settings.
3. Go to Pages.
4. Set the source to deploy from the main branch and the root folder.
5. Save, then open the GitHub Pages URL once GitHub finishes deployment.

## Launch notes

Before sharing publicly with customers, replace placeholder pricing with owner-approved pricing. This static GitHub Pages version opens the visitor's email client with the order details prepared and provides a Facebook DM handoff with copyable details. Fully automatic background email and messaging will require a small backend or third-party form service.
